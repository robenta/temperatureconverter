Temperature Converter

A simple web application built with HTML, CSS, and JavaScript that converts temperatures between Kelvin, Celsius, and Fahrenheit. This project helped me practice working with JavaScript functions and DOM manipulation.

Features:

Convert temperatures between Celsius, Fahrenheit, and Kelvin

Instant conversion results as you input values

Simple, clean interface styled with CSS

Responsive layout for desktop and mobile

Purpose:
The project was created to practice JavaScript calculations, handling user input, and updating the webpage dynamically.
